# DeliveryService Manager: Система учета услуг доставки продуктов
## Описание
#"DeliveryService Manager" — это консольное приложение, предназначенное для автоматизации учета услуг фирмы по доставке продуктов. Система позволяет вести базу клиентов, оформлять заказы, отслеживать статусы доставки, формировать отчеты и анализировать активность клиентов. Приложение разработано в рамках учебной практики в Пензенском колледже информационных и промышленных технологий (ИТ-колледж).

## Основные возможности

- Управление клиентами: Добавление, редактирование, удаление клиентов

- Управление заказами: Создание заказов, назначение курьеров, изменение статусов

- Отслеживание статусов доставки: Оформлен → В процессе → Доставлен/Отменён

- Формирование отчетов: Ежедневные, еженедельные, ежемесячные отчеты

- Экспорт данных: Поддержка форматов JSON и CSV

- Анализ статистики: Количество заказов, выручка, активные клиенты

## Использование
### Для администраторов:
1. Авторизуйтесь в системе как администратор

2. Управляйте клиентами (добавление/редактирование/удаление)

3. Создавайте и назначайте заказы

4. Формируйте отчеты и анализируйте статистику

5. Экспортируйте данные в JSON/CSV форматах

## Для курьеров/операторов:
1. Авторизуйтесь в системе как курьер

2. Просматривайте назначенные заказы

3. Изменяйте статусы заказов (В процессе/Доставлен/Отменён)

4. Просматривайте контактные данные клиентов
