# QuoteKeeper: Персональный каталог цитат из книг

## Описание

**QuoteKeeper** — это приложение с графическим интерфейсом (Windows Forms) для ведения личного каталога цитат из книг. Программа позволяет добавлять, искать, удалять и экспортировать цитаты. Проект разработан в рамках учебной практики в Пензенском колледже информационных и промышленных технологий (ИТ-колледж).

## Основные возможности

- **Добавление цитат:** Ввод автора, названия книги, текста цитаты и темы
- **Просмотр всех цитат:** Отображение списка всех сохранённых цитат
- **Поиск по автору:** Поиск цитат по имени автора (частичное совпадение)
- **Поиск по теме:** Поиск цитат по теме (частичное совпадение)
- **Случайная цитата («Цитата дня»):** Показ случайной цитаты из каталога
- **Удаление цитат:** Удаление выбранной цитаты из списка
- **Сохранение и загрузка:** Автоматическое и ручное сохранение данных в файл
- **Экспорт в JSON:** Выгрузка всех цитат в файл формата JSON

# Использование
## Главное окно программы состоит из трёх вкладок:
### Вкладка «Управление цитатами»
- Заполните поля: Автор, Книга, Цитата, Тема

- Нажмите «Добавить цитату» для сохранения

- Справа отображается список всех цитат

### Вкладка «Поиск»
- Введите имя автора и нажмите «По автору» для поиска

- Введите тему и нажмите «По теме» для поиска

- Результаты отображаются во всплывающем окне

### Вкладка «Действия»
- Все цитаты — показать весь список

- Цитата дня — показать случайную цитату

- Удалить цитату — удалить выбранную цитату

- Сохранить — вручную сохранить данные в файл

- Загрузить — вручную загрузить данные из файла

- Экспорт в JSON — выгрузить все цитаты в JSON-файл

# Технические детали
- Язык программирования: C#

- Тип приложения: Windows Forms (графический интерфейс)

- Архитектура: Объектно-ориентированное программирование (ООП)

- Хранение данных: Текстовый файл quotes.txt (формат: Автор|Книга|Цитата|Тема)

- Экспорт данных: JSON (файл quotes_export.json)

- Среда разработки: Visual Studio Code / Visual Studio

- Система сборки: dotnet build / MSBuild

# Особенности реализации
- Контроль корректности вводимых данных (проверка на пустые поля)

- Обработка ошибок ввода-вывода при работе с файлом

- Автоматическое создание файла при его отсутствии

- Восстановление работоспособности при повреждении файла

- Автосохранение после каждого добавления или удаления

- Уникальные идентификаторы (ID) для каждой цитаты
